📋 How to Play

  1. One person creates a game and shares the 4-letter room code.
  2. Everyone joins with the code on their own phone.
  3. Each round has 5 trivia questions about women's professional soccer.
  4. You have 10 seconds to pick A, B, or C — unanswered = wrong!
  5. After each of the 3 rounds, the player in last place is eliminated.
  6. After Round 3, the player with the most correct answers wins! 🏆
  7. If there's a tie for 1st, a sudden-death tiebreaker decides it all.
⚡ +1 point for every correct answer — speed doesn't matter, accuracy does!
